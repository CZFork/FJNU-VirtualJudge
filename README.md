# FJNU-OnlineJudge

基于Laravel与Swoole搭建的ACM-ICPC在线评测平台~

目前正在加紧开发WEB端功能，预计4月份放出虚拟评测完整应用，已经支持HDOJ,POJ,FOJ三大主流评测平台。
